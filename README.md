# 🌾 Kissan Sarthi — AI-Powered Smart Farming Assistant

An end-to-end AgriTech web application built with Flask and AI, 
designed to empower Indian farmers with intelligent crop management tools.

## 🚀 Features

| Feature | Description |
|---|---|
| 🌿 Plant Disease Detection | Custom 8-layer CNN trained on 39 disease classes |
| 🎤 AI Voice Assistant | Gemini API powered assistant with mic + image input |
| 🌍 Multilingual Support | English, Hindi, and Punjabi |
| ☁️ Weather Advisory | Live weather API with AI crop recommendations |
| 📋 Crop Guide | Day-0 to harvest guide with downloadable PDF |
| 💰 Economy Calculator | Investment, revenue & profit calculator with report |

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **ML/DL:** TensorFlow, Keras, CNN (8-layer, 39 classes)
- **AI:** Google Gemini API
- **APIs:** Weather API, Speech Recognition
- **Frontend:** HTML, CSS, JavaScript
- **Other:** gTTS, ReportLab (PDF generation)
